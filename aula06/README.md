*Perguntas/Dúvidas e Respostas:

Qual a diferença entre == e ===?
    == é usado para comparar se um número é igual ao outro e === é usado para comparar se um número é igual e idêntico ao outro. O === compara tanto o valor quanto o tipo, exemplo, string ou number.

Como fazer a entrada de dados em JS?
    Pelo DOM ou PROMPT com a interface gráfica, ou seja, a entrada de dados tem que ser feita pelo navegador, Google, Edge, etc.

Qual a diferença entre != e !?
    != é comparação
    ! é negação/inversão

Como identificar Media queries em JS?
    Use dentro de if "navigator.userAgent.match(/Android/i)" com o modelo dentro dos parentêses e use else para caso não seja nenhuma das opções e mudar a interface para o PC. Ps: estudar mais sobre "navigator.userAgent".

O que é o DOM?
    É a árvore de elementos do HTML.
    window
        -> location
        -> document
            -> html
                -> head
                    -> meta
                    -> tittle
                -> body
                    -> h1
                    -> p
                        -> strong
                        -> span
                    -> div
                    -> main
                    -> a
        -> history

Defer no script
    O defer faz com que o "script" carregue em paralelo com o HTML.
