*Operadores Lógicos em JavaScript

&& -> and/e
|| -> or/ou
! -> not/não

**Tabela Verdade

AND/E 
    --> V COM V -> V / 1 COM 1 -> 1
    --> V COM F -> F / 1 COM O -> 0
    --> F COM V -> F / 0 COM 1 -> 0
    --> F COM F -> F / 0 COM 0 -> 0

OR/OU 
    --> V COM V -> V / 1 COM 1 -> 1
    --> V COM F -> V / 1 COM 0 -> 1
    --> F COM V -> V / 0 COM 1 -> 1
    --> F COM F -> F / 0 COM 0 -> 0

ATALHO: Segurando Ctrl + ; comenta todo o conteúdo selecionado.
