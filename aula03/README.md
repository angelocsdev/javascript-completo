*VAR, LET E CONST

Pode criar quantas variáveis foram necessárias em um programa.

VAR é usado quando quer uma variável que pode ser acessada de qualquer lugar sem restrições.

LET é usado para executar variáveis do mesmo bloco ou blocos dentro do bloco em que let está. Blocos/escopos antes de let não dão certo, eles têm que estar no mesmo nível.

CONST é usado para que o valor da variável não possa mais ser alterado.
