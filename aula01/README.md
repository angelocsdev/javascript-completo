Anotações Principais:

O comentário de 1 linha é feito com //

E do de vários linhas com /**/

Use a tag 'script:src' para vincular um arquivo JS a um HTML. PS: evitar colocar o script dentro do head porque isso fará com que ele seja executado antes de algumas coisas na página que ele deveria ler causando assim alguns bugs. Então é recomendado que seja colocado no final do body.

A extensão para o node.js funciona apertando F8.
