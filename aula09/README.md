#Pré e Pós Incremento

Pós incremento é quando usa a variável e depois acrescenta o valor:
Ex: console.log(n++), nesse caso o valor quando executado vai ser o mesmo, mas quando rodar de novo vai adicionar 1.

Pré incremento é quando usa a variável e incrementa o valor ao mesmo tempo:
Ex: console.log(++n), vai adicionar 1 ao mesmo tempo que executar sem precisar de um outro comando.

Ps: A mesma coisa funciona no decremento -> n-- / --n
