*Operadores Relacionais

Operadores relacionais são a mesma coisa que operadores de comparação.

Eles são:

    > -> maior
    >= -> maior ou igual
    < -> menor
    <= -> menor ou igual
    == -> igual
    != -> diferente

Atenção ao igual!

Um igual é usado para atribuição, para dar um valor a uma variável.
Dois iguais são usados para comparação.
