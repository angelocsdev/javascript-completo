*Operadores Matemáticos/Aritméticos em JavaScript:

+ -> soma
- -> subtração
/ -> divisão
* -> multiplicação
% -> resto de divisão
** -> potência
++ -> incremento
-- -> decremento
+= -> outra forma de incremento
-= -> outra forma de decremento

Ordem de precedência -> primeiro resolve entre (), depois ** -> potência, depois * -> multiplicação e % -> divisão de resto e / -> divisão normal, e depois + -> adição e - -> subtração.
